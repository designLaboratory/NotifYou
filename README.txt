# WS2812-light-LED
Base functionality of WS2812 LEDs

readme.txt
----------
This is a project for uVision 5 to drive the WS2812 (Adafruit NeoPixel) LEDs using NXP Kinetis KL46Z microcontroller board.

The project contains a header files and main.c file which is presenting base functionality of a program. It is neopixels as a array of LEDs using TPMs and DMA transfers
Output of the signal is on PTB2.
